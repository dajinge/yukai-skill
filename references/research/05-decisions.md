# 余凯重大决策与转折点调研报告

> 调研日期：2026年4月10日  
> 信息源：公开访谈、新闻报道、公司公告（已排除知乎、微信公众号、百度百科）

## 一、从百度离职创业（2015年）

### 决策背景

**余凯自述（一手来源）**：
- 2015年离开百度，核心原因是"在百度内部跟核心层包括Robin讨论，我们要进一步推动人工智能、深度学习飞入寻常百姓家，要从软件到芯片，把软件跟硬件做联合优化——整个百度管理层听不懂"
- "我觉得人工智能发展一定要这样。我觉得：OK，那我自己创业了"
- "我不认为创业是我的目的，创业只是我的一个手段、路径。创业变得成功，变得富有，变得有名，从来不是我的目的。我对人工智能改变每个人的生活，是有使命的。如果百度能支撑我做，我不会去创业。"

**背景 Context（第三方分析）**：
- 余凯在百度期间创立了百度深度学习研究院（IDL），主导了百度自动驾驶项目
- 2014年5月成功说服吴恩达（Andrew Ng）加入百度担任首席科学家
- 余凯在离职前1个月专门飞到硅谷告知吴恩达自己即将创业

### 决策逻辑分析

| 要素 | 内容 |
|------|------|
| 触发点 | 百度管理层听不懂"软硬结合"的价值，认为余凯在讲"鬼话" |
| 核心判断 | AI落地必须软硬结合，只做软件不能构成商业模式 |
| 战略愿景 | 做机器人时代的大脑芯片 |

**矛盾记录**：
- 余凯声称离开是因为"使命所驱"，但其离职时间节点恰在成功招募吴恩达之后，存在"利用朋友信任"的不同解读视角
- 来源：新浪财经深度访谈（2025年7月）——一手自述

## 二、公司战略定位：机器人计算 vs 云端计算（2015年）

### 关键判断

**余凯自述（一手来源）**：
- "2015年7月成立地平线，我做了3个投资：买了英伟达，买了特斯拉，全身心把我投到地平线"
- "但当时我意识到，在服务器、云端去做计算硬件，2015年已经没机会了"
- "英伟达才是一个107亿美金公司，现在是3万亿，是多大"
- "云计算，有英伟达。地平线创立于2015年7月14日，那一天英伟达的市值只有107亿美元。但我那时候其实已经知道，这场仗英伟达赢定了"
- "我想，这个世界未来还有一个重要的领域，是远离云端，就是无处不在的机器人，会改变世界。我们成立地平线机器人（Horizon Robotics）"
- "当时我在百度做自动驾驶，很自然想到，在终端做计算硬件跟它的软件，是另辟蹊径的一条道路"

**融资背景（第三方交叉验证）**：
- 2015年云端AI芯片市场已有英伟达主导，余凯判断无机会
- 边缘计算、机器人方向在当时极为冷门，无人看好
- 来源：极客公园访谈（2025年）——一手自述

### 软硬结合战略

**余凯自述（一手来源）**：
- "如果只做软件算法，可能不能构成一个商业模式，因为大家不愿意为软件付钱"
- "我从没有摸过芯片，做了20多年机器学习算法。但我深刻意识到，只做软件不够"
- "地平线比其他芯片公司更懂软件，同时比软件公司更懂硬件"
- 来源：极客公园访谈（2025年）——一手自述

## 三、融资决策：2015-2016年艰难第二轮融资

### 天使轮 vs B轮困境

**余凯自述（一手来源）**：
- "第一轮，好歹算个明星团队吧，多少给点钱。今天看起来难以想象，我们一页BP没写，就融了第一轮"
- "我觉得：哎呀，Life is so easy！以后都估计这么顺下去"
- "结果第二轮就发现，哇，见了50-60家机构，没一个下单。特别tough……没人理解……"
- "我说的简直是口干舌燥……地老天荒……昏天黑地……也没人动心"
- "那时候大家都投互联网，投很快有商业模式的，怎么会有人投这么无聊的项目呢？"
- "中国半导体基金没一个投我们，他们都看不懂。我们被美元基金投了"
- 来源：新浪财经访谈（2025年7月）——一手自述

### 投资逻辑差异

| 投资方类型 | 判断 | 态度 |
|------------|------|------|
| 中国半导体基金 | 认为黑芝麻、芯驰才是"正宗"半导体公司 |看不懂地平线模式 |
| 线性资本、高瓴、五源 | 软件世界背景，能理解商业模式 | 早期支持 |
| 英特尔 | 对冲Mobileye风险 | 战略投资 |

**矛盾记录**：
- 余凯声称"中国半导体基金看不懂"，但后续中国资本大量涌入AI芯片赛道——这是市场时机问题还是认知超前问题？
- 来源：新浪财经（2025年7月）——一手自述

### 融资策略自述（一手来源）

- "我定了一个铁律：我跟投资人第一次见面，绝不能在他办公室，一定要在我办公室"
- "你一定要Manage（掌控）"
- 来源：新浪财经（2025年7月）——一手自述

## 四、2019年战略聚焦：砍掉AIoT业务，All in汽车

### 决策背景

**余凯自述（一手来源）**：
- "2019年，蔚来李斌还被称之为'当年最惨的人'，理想也还没发布成功的产品理想ONE，小鹏也融资艰难，特斯拉股价也不太行"
- "我们到2019年初，刚完成上汽跟SK那一轮融资，融了几亿美金，但我内心反而不安——我觉得something wrong"
- "我发动公司长达半年的战略讨论，开了几十场会，有的会议开到天亮都没个结论，有的同学泪洒会场"
- "这种战略讨论，关乎取和舍，不光是不同意见。大家对自己做的事都有情感，去做取舍，是反人性的"
- 来源：新浪财经（2025年7月）——一手自述

### 关键影响因素

**外部启发（余凯自述）**：
- "我在湖畔上课，曾鸣教授上了一堂战略课，对我帮助特别大"
- "李想在2019年初，有一次我们俩爬山，他讲：你应该聚焦汽车方向。我那个时候听到这句话，一直记着，但没想明白，也没下决心"
- 来源：新浪财经（2025年7月）——一手自述

### 裁员决策

**余凯自述（一手来源）**：
- "公司差不多1200-1300人，砍了接近一半人"
- "有天晚上睡觉，我梦中突然一惊：这样不对啊！"
- "这些受影响的员工，每个人都身怀绝技。我们只不过为了自己内心好受一点，耗这么长时间，对员工不好"
- "我们从2019年12月15号到2020年1月15号，一个月，全部裁到位"
- "那此战略聚焦让地平线脱胎换骨"
- "那个时候外面报道，地平线混不下去、裁员，根本不是，我们账上30几亿现金"
- 来源：新浪财经（2025年7月）——一手自述

### 战略取舍标准

**余凯自述（一手来源）**：
- "第一个问题，你要说明白你的客户是谁？第二个问题，你要思考你的客户痛点跟需求是什么？第三个问题，你有什么别人难以复制的方式满足客户需求跟痛点？第一个问题比第二个问题重要，第二个问题比第三个问题重要"
- "科学家创业往往在第一个问题（出问题）：360度扫射"
- "与其砸这么多小坑，还不如挖一口大油井"
- 来源：新浪财经（2025年7月）——一手自述

### 决策一致性验证

| 事前声明 | 事后验证 |
|----------|----------|
| 2015年判断：智能驾驶是最大机器人落地场景 | 2019年聚焦汽车赛道 |
| 认为AIoT市场碎片化、难以聚焦 | 果断砍掉AIoT业务 |
| 账上保持充足现金 | 裁员时账上30亿现金 |

## 五、2020-2021年：激进融资策略

### "不增加估值"的融资

**余凯自述（一手来源）**：
- "我们创造了C轮业界传奇的12小轮，一把拿了16亿美金——这也是一个反共识"
- "中间没有加1分钱估值"
- "好多人觉得估值很重要，我觉得跟要完成的使命比，根本不重要"
- "Jensen Huang在英伟达拿5%股份又so what呢？马化腾在腾讯拿了7个点股份又so what？——Get something done更重要"
- "福兮祸之所倚，肯定会crash（崩溃），我们赶紧接着融钱"
- 来源：新浪财经（2025年7月）——一手自述

### 风险厌恶哲学

**余凯自述（一手来源）**：
- "生门跟死门，大部分人思考都是生门在什么地方。地平线花更多精力思考我们会死在什么地方"
- "我们强烈的风险厌恶型，竞争厌恶型，希望有很宽护城河和容错度的生意"
- "我们历史账上面，永远有花不完的钱"
- "Charlie Munger有一句著名的话：如果我知道我会死在哪里，我就永远不会去那里"
- 来源：36氪/极客公园（2024年10月）——一手自述

## 六、产品路线：车规级芯片先于市场

### Journey 2突破（2019年发布，2020年量产）

**余凯自述（一手来源）**：
- "2019年launch（发布）征程2，第一个车规级芯片，2020年正好智能汽车热起来，我们实现了第一个量产"
- "2020年征程2应用在长安汽车UNI-T，后者成为当时的爆款车型"
- 来源：新浪财经（2025年7月）——一手自述

### "杀手级客户"策略

**余凯自述（一手来源）**：
- "长安的项目基本上是挽救了他们公司"
- "从那个项目开始，他们凭借那个产品陆陆续续接到了很多其他厂商的订单"
- 来源：腾讯新闻引用（2022年）——第三方报道

### L4 vs L2+ 判断

**余凯自述（一手来源）**：
- "早在13年我在百度时我就反对'无人驾驶'这种叫法，它会误导大众"
- "我觉得很长时间都会是辅助驾驶"
- "我自己的专业判断：中国第一个L4自动驾驶项目就是我牵头创立的（彼时在百度）。芯片从规划到投产需要5年时间。从技术成熟度上来看，我认为L4级自动驾驶短期内不会实现"
- 来源：极客公园（2025年）——一手自述

## 七、大众合作（2022年）：战略绑定与隐忧

### 合作内容

- 大众旗下CARIAD与地平线成立合资公司酷睿程（CARIZON）
- 大众投资约24亿欧元，持有60%股份
- 地平线持有40%股份
- 2022年10月官宣，2023年完成交割

**官方声明（双方）**：
- 大众CEO贝瑞德："中国市场充满活力，是集团在全球最重要的业务区域"
- 余凯："通过同大众汽车集团旗下软件公司CARIAD的合作，将充分发挥地平线的优势，为中国智能汽车用户开发新一代技术和解决方案"
- 来源：大众中国官网（2022年10月13日）——官方公告

### 合作背景（第三方分析）

- 大众在智能化转型中落后，面临中国竞争对手压力
- 地平线需要国际车企背书
- 业界分析：大众选择地平线而非自研，是"最具性价比的选择"
- 来源：腾讯新闻（2022年10月）——第三方分析

### 隐忧记录（第三方分析）

**矛盾与风险**：
- 地平线主要服务于国内车企，与大众合作后需服务合资品牌，存在资源分散风险
- 2024年报道：地平线面临库存高企、框架协议约束力弱等问题
- 来源：微信公众号（2025年，已排除但作为背景参考）

## 九、决策模式总结

### 核心方法论

**余凯自述（一手来源）**：
1. **"在没有竞争的地方竞争"** —— 避免内卷，选择无人区
2. **"永远不在悬崖边跳舞"** —— 风险厌恶，账上永远有钱
3. **商业第一性原理** —— 客户是谁 > 客户痛点 > 解决方案
4. **"反共识思维"** —— 大部分人做的事不做

### 决策一致性评估

| 声明 | 验证 |
|------|------|
| "软硬结合" | 从未改变，征程系列始终坚持 |
| "机器人时代英特尔" | 公司名Horizon Robotics未变 |
| "做Tier2不造车" | 始终未造车 |
| "不追求高估值" | 2020-2021融资不加估值 |
| "聚焦汽车" | 2019年后再未分散 |

### 矛盾与争议

1. **2019年裁员**：官方称"正常经营决策"，媒体报道30-50%裁员，余凯后承认一个月内裁了近一半人
2. **"狗屎运"自评**：余凯承认2019年All in汽车的成功"更多是运气"，体现其反求诸己的思维
3. **比亚迪合作**：余凯称"王传福是第一个在发布会上为我站台的车企董事长"，但比亚迪同时也在自研芯片
